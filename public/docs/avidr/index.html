How to Login to Service Engine in Tanzu

Steps:

  1. Login to the bastion host.

  2. Login to the Avi Controller (Use the IP address of one of the Avi Controllers):

ssh admin@10.252.240.50

Use the same username and password as the Avi Controller login.

  1. Enter the Avi CLI shell:

admin@avi-controller-dr-1-marc-fr-ssg:~$ shell

If prompted, provide credentials:

Login: admin Password:

  1. Attach to the Service Engine:
  • List available service engines:

attach serviceengine Avi-se- Avi-se-himsz Avi-se-tgkld

  • Attach to a specific service engine (example: Avi-se-himsz):

attach serviceengine Avi-se-himsz

You may see a warning like:

Warning: Permanently added ‘[127.1.0.1]:5097’ (ECDSA) to the list of known hosts.

Notes:

  • The serviceengine command works only inside the Avi CLI shell, not in the normal bash shell.
  • If you get command not found, make sure you are inside the Avi CLI shell by running shell and logging in.
  • Use the attach serviceengine command to connect to a specific Service Engine by its name.
