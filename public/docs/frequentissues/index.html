Frequent issues

If you get below error in teams Monitoring

[FIRING:1] (NodeFilesystemSpaceFillingUp /dev/mapper/vstorage-gitlablog ext4 10.6.39.25:9100 node /var/log/gitlab monitoring/k8s warning)

Solution

sudo /opt/gitlab/embedded/sbin/logrotate -fv /var/opt/gitlab/logrotate/logrotate.d -s /var/opt/gitlab/logrotate/logrotate.conf

2 [FIRING:1] (NodeFilesystemSpaceFillingUp /dev/mapper/rhel-var ext4 10.6.39.25:9100 node /var

sudo find /var/opt/gitlab/gitlab-rails/shared/cache/archive -type f -delete

Find out the files

find /var -type f -exec du -h {} + | sort -rh | head -n 10

Delete files which are older than 30 days

find /var/log -type f -name "*.gz" -mtime +30 -exec rm {} \;
